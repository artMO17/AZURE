# AZURE
Este repositorio ha sido creado con el objetivo de centralizar, documentar y estructurar todo el proceso de aprendizaje relacionado con los servicios de Microsoft Azure, especialmente en el marco de la certificación Microsoft Azure Administrator (AZ-104).

A lo largo de este espacio iré recopilando:

- Prácticas realizadas en el portal de Azure
- Ejercicios guiados y laboratorios
- Scripts utilizados durante las sesiones
- Notas técnicas, resúmenes y conceptos clave
- Casos prácticos y escenarios de administración en la nube
- Buenas prácticas y conclusiones derivadas de cada módulo

## Propósito del repositorio
El propósito principal de este repositorio es apoyar mi proceso de estudio y consolidación de conocimientos sobre la administración de recursos en Azure, utilizando un enfoque práctico y organizado.
